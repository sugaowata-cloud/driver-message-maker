# 運転連絡メーカー

送迎業務で使うLINE向け定型文を、スマートフォンで作成・編集・コピーする静的PWAです。データはすべて端末の `localStorage` に保存され、外部サービスには送信されません。

## ファイル構成

```text
driver-message-maker/
├─ index.html
├─ style.css
├─ app.js
├─ manifest.json
├─ service-worker.js
├─ icon.svg
└─ README.md
```

## ローカルで起動する

PWAとコピー機能を正しく試すには、ファイルを直接開かずローカルサーバーを使います。

Pythonがある場合、`driver-message-maker` フォルダで次を実行してください。

```sh
python -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開きます。同じWi-Fi上のスマートフォンから確認する場合は、PCのローカルIPアドレスを使って `http://PCのIPアドレス:8000/` を開きます。なお、スマートフォンでClipboard APIやPWAを完全に使うにはHTTPSでの公開が必要です。

## GitHub Pagesへ公開する

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内の全ファイルをリポジトリ直下へ追加してpushします。
3. リポジトリの「Settings」→「Pages」を開きます。
4. 「Build and deployment」で「Deploy from a branch」を選びます。
5. 公開ブランチを `main`、フォルダを `/(root)` にして保存します。
6. 表示された `https://ユーザー名.github.io/リポジトリ名/` を開きます。

相対パスで構成しているため、GitHub Pagesのサブディレクトリでも動作します。

## スマートフォンのホーム画面へ追加する

### iPhone / iPad（Safari）

公開URLをSafariで開き、「共有」→「ホーム画面に追加」→「追加」を選びます。

### Android（Chrome）

公開URLをChromeで開き、メニューの「ホーム画面に追加」または「アプリをインストール」を選びます。

## テンプレート文面を変更する

`app.js` 冒頭の `MESSAGE_TEMPLATES` で、各定型文のタイトル・アイコン・文頭・文末を変更できます。文章の組み立て方は同ファイルの `generateMessage()` にまとまっています。

自由テンプレートは、アプリの「自由テンプレ」→「新規作成」から追加できます。`{{日付}}`、`{{時刻}}`、`{{場所}}`、`{{到着時刻}}`、`{{所要時間}}`、`{{名前}}`、`{{URL}}` を使用できます。

## 保存とプライバシー

- デフォルト送信先、文末設定、絵文字設定
- 場所候補（各10件）
- 自由テンプレート
- 生成履歴（直近10件）

上記はブラウザ内だけに保存されます。ブラウザのサイトデータを消すと失われます。履歴は文面を作成・コピーした記録であり、LINEへの送信証明ではありません。

## 制限事項

- LINEへの自動送信や送信先の自動選択は行いません。
- 位置情報の自動取得、ログイン、クラウド同期、通知には対応していません。
- Web Share APIは対応端末でのみ表示されます。
- Clipboard APIはHTTPSまたはlocalhostで利用し、使えない場合は従来方式へ自動で切り替えます。
- 初回のオフライン利用前に、オンライン状態で一度アプリを開いてキャッシュを作る必要があります。
